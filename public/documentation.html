Add an user

/users

POST

A JSON object holding data about the user to add, structured like:

{

"Username": "karolina2000",

"Password": "street123",

"Email": "karolina1997@gmail.com",

"Birthday": "1992-07-07"

}

A JSON object holding data about the user that was added, including an ID:

{

"Username": "karolina2000",

"Password": "street123",

"Email": "karolina1997@gmail.com",

"Birthday": "1992-07-07T00:00:00.000Z",

"FavouriteMovies": [],

"_id": "655e3221b4280b8bf525ba6d",

"__v": 0

}
